### Usage Examples

**CDN (UMD):**
```html
<script src="https://unpkg.com/Shiply-sdk@1.0.0/dist/Shiply.min.js"></script>
<script>
  const Shiply = new Shiply({
    apiKey: 'your-api-key',
    websiteId: 'your-website'
  });
  Shiply.init().show();
</script>
```

**NPM (ES Modules):**
```javascript
import Shiply from 'Shiply-sdk';

const Shiply = new Shiply({
  apiKey: 'your-api-key',
  websiteId: 'your-website'
});
Shiply.init().show();
```

**NPM (CommonJS):**
```javascript
const Shiply = require('Shiply-sdk');

const Shiply = new Shiply({
  apiKey: 'your-api-key',
  websiteId: 'your-website'
});
Shiply.init().show();
```
